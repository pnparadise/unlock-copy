# Unlock Copy Paste - Chrome Extension

专注于解锁网站复制限制的Chrome扩展程序，Ctrl+C 复制功能进行了深度优化。

## 功能特性

- 🔓 **专业复制解锁**：使用先进的 copy 事件拦截技术，彻底绕过网站复制限制
- 📋 **智能复制处理**：自动检测并处理各种复制场景，确保复制成功
- 🎯 **Toast 提示**：复制成功时显示友好的提示信息
- 🔄 **开关控制**：支持一键启用/禁用扩展功能
- 📱 **状态指示**：工具栏徽章实时显示扩展状态
- ✨ **文本选择**：自动启用被禁用的文本选择功能
- 🎨 **现代界面**：简洁美观的中文用户界面

## 核心技术

### 高级 Copy 事件拦截
```javascript
// 在捕获阶段拦截所有 copy 事件
document.addEventListener('copy', function(e) {
    e.stopImmediatePropagation(); // 阻止网站处理
    // 执行我们自己的复制逻辑
}, true);
```

### 多重复制保障
1. **Clipboard API**：现代浏览器的首选方法
2. **execCommand 备用**：兼容旧版浏览器
3. **Textarea 方法**：终极备用方案

## 安装方法

### 开发者模式安装

1. 下载或克隆此项目到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启右上角的"开发者模式"

## 使用方法

### 基本使用
1. 安装后扩展自动激活
2. 在任何网站选中文本
3. 按 `Ctrl+C` 即可复制
4. 复制成功会显示绿色提示

- **开关按钮**：在弹窗右侧可快速切换

## 工作原理

### 1. Copy 事件拦截
- 在捕获阶段拦截所有 copy 事件
- 使用 `stopImmediatePropagation()` 阻止网站处理
- 立即执行我们的复制逻辑

### 2. 智能复制策略
```javascript
// 优先使用 Clipboard API
if (navigator.clipboard) {
    await navigator.clipboard.writeText(text);
} else {
    // 备用 execCommand 方法
    document.execCommand('copy');
}
```

### 3. 状态管理
- 扩展状态持久化存储
- 实时同步工具栏徽章
- 支持多标签页状态同步

## 常见问题

### Q: 复制后没有提示怎么办？
A: 检查扩展是否启用，工具栏图标不应显示红色 "OFF" 徽章。

### Q: 某些网站仍无法复制？
A: 尝试刷新页面，或在扩展弹窗中重新启用扩展。

### Q: 如何确认复制成功？
A: 复制成功会在页面右上角显示绿色 "✅ 已复制 X 个字符" 提示。

### Q: 扩展会影响网站功能吗？
A: 不会。扩展只处理复制相关功能，不干扰网站其他操作。
